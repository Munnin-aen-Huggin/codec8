## Recording Notes

1. **Prep:** Have a clean test repository with actual code (not empty)
2. **Speed:** Don't rush - let viewers see the UI
3. **Cursor:** Use slow, deliberate mouse movements
4. **Audio:** Record in quiet environment, use external mic if available
5. **Resolution:** Record at 1080p or higher
6. **Editing:** Cut long loading times, keep transitions smooth
7. **Music:** Consider subtle background music (royalty-free)

## Technical Requirements

- Screen recording: OBS, Loom, or ScreenFlow
- Video editing: Basic cuts in any video editor
- Export: MP4, H.264, 1080p, 30fps minimum
- File size: Aim for under 100MB for easy uploads

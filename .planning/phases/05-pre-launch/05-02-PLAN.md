# Phase 5.2: Launch Assets

## Objective
Create compelling marketing assets for Product Hunt launch: demo video, screenshots, and launch copy.

## Prerequisites
- App stable (after beta fixes)
- Representative test data in production
- Screen recording software available

## Tasks

### 1. Demo Video Script
**Type:** Documentation

- [ ] Write 2-3 minute video script
- [ ] Structure: Problem (30s) -> Solution demo (90s) -> CTA (30s)
- [ ] Key scenes:
  - Landing page overview
  - GitHub OAuth login
  - Connect a repository
  - Generate documentation (all 4 types)
  - View/edit generated docs
  - Show pricing tiers

**Acceptance:** Script document ready for recording

### 2. Demo Video Recording
**Type:** Content Creation

- [ ] Set up screen recording (OBS, Loom, or similar)
- [ ] Record clean demo with script
- [ ] Add voiceover or captions
- [ ] Edit for pacing (cut pauses, mistakes)
- [ ] Export at 1080p minimum

**Acceptance:** 2-3 minute polished demo video

### 3. Screenshot Assets
**Type:** Content Creation

- [ ] Landing page hero shot (1200x630 for social)
- [ ] Dashboard with repos (show value)
- [ ] Doc generation in progress (show AI working)
- [ ] Generated README preview (show quality)
- [ ] Pricing section (show value proposition)
- [ ] Before/after comparison (optional)

**Acceptance:** 5+ high-quality screenshots, properly sized

### 4. Product Hunt Submission Draft
**Type:** Documentation

- [ ] Tagline (60 chars max)
- [ ] Description (260 chars max)
- [ ] Longer description (500+ chars)
- [ ] Maker comment draft
- [ ] Topics: Developer Tools, Artificial Intelligence, Documentation
- [ ] First comment strategy

**Acceptance:** All PH fields drafted and reviewed

### 5. Launch Copy
**Type:** Documentation

- [ ] Twitter/X launch thread (5-7 tweets)
- [ ] Hacker News Show HN post
- [ ] Reddit post for r/SideProject
- [ ] Email to beta users announcing launch

**Acceptance:** All launch copy written and ready to post

## Checkpoints

### Checkpoint 1: Script & Screenshots Ready
- Video script finalized
- All screenshots captured
- Basic assets organized

### Checkpoint 2: Video Complete
- Demo video recorded
- Video edited and exported
- Uploaded to hosting (YouTube unlisted, Loom, etc.)

### Checkpoint 3: All Copy Ready
- Product Hunt submission drafted
- Social media posts written
- Email templates ready

## Estimated Scope
- Video script: Small (1 hour)
- Video recording/editing: Medium (3-4 hours)
- Screenshots: Small (1 hour)
- Product Hunt copy: Small (1-2 hours)
- Launch copy: Small (1-2 hours)

## Dependencies
- Depends on 05-01 (beta fixes complete for stable demo)

## Risks
- Video quality issues: Do multiple takes, use good mic
- App changes after recording: Record near launch date
